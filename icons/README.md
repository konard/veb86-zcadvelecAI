# Иконки Grid 3x2

Набор из 5 SVG-иконок с сеткой 3x2 (3 строки по вертикали, 2 колонки по горизонтали).

## Спецификации

- **Размер**: 16x24 пикселя
- **Цвет границ**: #20A8FF (голубой)
- **Цвет заливки**: #008C00 (зелёный)
- **Толщина границ**: 1.2 пикселя
- **Стиль**: Округлые края и соединения линий

## Варианты иконок

### grid_icon_01.svg
Все 6 ячеек заполнены зелёным цветом. Полностью заполненная сетка.

### grid_icon_02.svg
Шахматный узор: заполнены 3 ячейки в шахматном порядке (верхняя левая, средняя правая, нижняя левая).

### grid_icon_03.svg
Заполнена левая колонка: все 3 ячейки слева заполнены, правая колонка пуста.

### grid_icon_04.svg
Заполнена верхняя строка: обе ячейки в верхнем ряду заполнены, остальные пусты.

### grid_icon_05.svg
Диагональный узор: заполнены 3 ячейки по диагонали (верхняя левая, средняя правая, нижняя правая).

## Использование

Эти иконки можно использовать в интерфейсах для обозначения различных состояний сеток, планировок помещений или других структурированных данных.

Все иконки созданы в соответствии с техническим заданием из issue #454.
